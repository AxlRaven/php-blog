# Логотип и обложка

## Через админку (рекомендуется)

**Настройки → Логотип и обложка**

| Файл | Путь на сервере | Назначение |
|------|-----------------|------------|
| Логотип | `uploads/images/logo.png` | Шапка сайта + favicon |
| Cover | `uploads/images/cover.jpg` | Картинка по умолчанию для OG / соцсетей |

Нужны права на запись в `uploads/images/`.

## Вручную (альтернатива)

Можно положить `logo.png` сюда (`assets/images/logo.png`) — он используется, если в uploads логотипа нет.

| Параметр | Значение |
|----------|----------|
| Формат | PNG / JPEG / WebP |
| Пропорции логотипа | квадрат 1:1 |
| Размер логотипа | **256×256 px** (рекомендуется) |

На сайте логотип показывается **36×36 px** (на телефоне 32×32).
